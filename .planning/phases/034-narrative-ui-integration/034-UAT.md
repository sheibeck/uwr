### 3. Create and List Hotbars
expected: Type "hotbar add buffs" — a new hotbar named "buffs" is created and becomes active. Type "hotbars" — shows a list of all your hotbars (at least "main" and "buffs") with their slot contents.

### 4. Switch Hotbars
expected: Type "hotbar main" to switch back to the main hotbar. The hotbar strip updates to show "main" as the active name. Click the right arrow — it switches to the next hotbar. Click left arrow — wraps back. The name label updates each time.
result: [pending]

### 5. Set Hotbar Slot
expected: Type "hotbar set 1 <ability_name>" (use an ability your character knows). Slot 1 on the active hotbar now shows that ability's name.
result: [pending]

### 6. Use Ability From Hotbar
