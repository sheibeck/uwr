# Plan 22-02: Rewrite all 16 class ability data files

## Goal
Replace every class ability file with the locked 6-ability progression (L1, L3, L5, L7, L9, L10), correct keys, descriptions, and stat values matching the Phase 22 design spec.

## Prerequisites
- Plan 22-01 (schema changes published) — not strictly required for data files, but publish must happen before the seeding reads these files
