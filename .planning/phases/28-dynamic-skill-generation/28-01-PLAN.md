**1. Create the unified dispatch map in combat.ts:**

Define an `AbilityActor` interface and `AbilityContext` type that works for both player characters and enemies:
```typescript
type AbilityActor = {
  type: 'character' | 'enemy';
  id: bigint;
  stats: { str: bigint; dex: bigint; int: bigint; wis: bigint; cha: bigint };
  level: bigint;
  name: string;
};

type AbilityRow = {
  kind: string; targetRule: string; value1: bigint; value2?: bigint;
  damageType?: string; scaling: string; effectType?: string;
  effectMagnitude?: bigint; effectDuration?: bigint; name: string;
  resourceType: string; resourceCost: bigint; cooldownSeconds: bigint;
};
```

Create `DISPATCH: Record<string, (ctx, combatId, actor, ability, targetCharacterId?, targetPetId?) => void>` with handlers for each kind. Port logic from the existing code:

- `damage`: Direct damage with armor/resist reduction (from the existing switch cases for damage abilities). Use `damageType` to determine physical vs magic resist. Scale with `scaling` stat.
- `heal`: Restore HP to target (from existing heal cases). Scale with `scaling` stat.
- `dot`: Apply damage-over-time effect using `effectType='dot'`, `effectMagnitude`, `effectDuration` (port from existing DoT handling).
- `hot`: Apply heal-over-time using effect system (port from existing HoT handling).
- `buff`: Apply positive effect with effectType, effectMagnitude, effectDuration (port from buff cases).
- `debuff`: Apply negative effect (port from debuff cases).
- `shield`: Apply damage_shield effect (port from existing shield handling).
- `taunt`: Add threat to caster (port from existing taunt mechanic).
- `aoe_damage`: Apply damage to all enemies (use THREAT_CONFIG.aoeDamageMultiplier). Scale value1 by aoe multiplier.
- `aoe_heal`: Heal all allies (port from existing aoe heal).
- `summon`: Create active_pet entry (port from existing summon logic).
- `cc`: Apply crowd control effect (stun/root/silence from effectType).
- `drain`: Deal damage and heal caster for portion (value1 = damage, value2 = heal%).
- `execute`: Bonus damage to targets below 30% HP (value1 = base damage, apply 2x multiplier below threshold).
- `utility`: Out-of-combat only effects (no combat action).

Create `resolveAbility(ctx, combatId, actor, ability, targetCharacterId?, targetPetId?)` as the single entry point. It looks up the handler from DISPATCH by `ability.kind`, throws if unknown.
